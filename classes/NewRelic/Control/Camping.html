Class NewRelic::Control::Camping < NewRelic::Control::Ruby

This is the framework control object for Camping apps. It is loaded by virtue of detecting ‘camping’ as the framework in the rpm_contrib/detection/camping.rb file. It gets loaded by the new_relic/control.rb file.
